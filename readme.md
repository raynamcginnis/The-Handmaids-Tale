# The Handmaid's Tale RPG Game

## How to Play

1.Pick YOUR Player
2.Pick your first opponent
3.Press the attack button to try and defeat your opponent
4.If you win, pick another opponent to attack
5.Blessed be the fruit!

- HTML

- Bootstrap
